# hobby_amazfit_bip_am_lich
App Âm lịch dành cho amazfit bip (BipOS)

# Cách dùng
1. Unpack file .res
2. Tải file `am_lich.elf` từ repo này và thêm vào res đã unpack
3. Pack file .res mới
4. Up lên bip và thưởng thức
